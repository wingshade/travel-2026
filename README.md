## 備用行程（雨天或彈性調整用）

### 高松市兒童未來館
- **位置**：高松市松島町1-15-1（高松站步行約9分）
- **特色**：室內親子設施，適合小孩放電
- **設施**：科學體驗區、遊戲區、工作坊
- **門票**：免費（部分體驗收費）
- **適合**：雨天、小孩需要放電時
- **時間**：建議停留2-3小時

### AEON MALL 綾川
- **位置**：JR綾川站步行約10分（從丸龜站約15分鐘車程）
- **特色**：四國最大購物中心
- **設施**：購物、美食街、遊戲區、電影院
- **營業**：10:00-21:00
- **適合**：雨天購物、用餐、休息
- **時間**：可停留半天

### 高松三越
- **位置**：高松站步行約15分（或琴電瓦町站步行3分）
- **特色**：百貨公司，有伴手禮、兒童樓層
- **營業**：10:00-19:30
- **適合**：最後採買、雨天購物
- **推薦**：地下1樓有讚岐烏龍麵、和菓子伴手禮

### 高松中央商店街
- **位置**：高松站步行約10分（與高松三越相連）
- **特色**：日本最長的有頂棚商店街（約2.7公里）
- **設施**：藥妝店、服飾店、餐廳、烏龍麵店、咖啡廳
- **營業**：各店鋪不同，多數10:00-20:00
- **適合**：雨天逛街、用餐、體驗當地生活
- **時間**：可停留1-2小時
- **推薦**：有頂棚完全不怕下雨！
